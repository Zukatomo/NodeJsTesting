Tesztelésben használt keretrendszerek
=======================================

A tesztelés során két keretrendszert használtam,
a `Mocha <https://mochajs.org/>`__ és `Chai <https://www.chaijs.com/>`__-t.

A tesztelés során a REST API néhány tulajdonságát fogom csupán tesztelni.
Ezek a tulajdonságok, az események lekérdezése különbőző paraméterek szerint,
illetve uj esemény hozzáadása és authentikáció.

Mocha
-----

A Mocha egy funkciókban gazad, tesztelést elősegitő
keretrendszer, amely NodeJS futtatókörnyezetben müködik, vagy
akár a böngészőben is `[1] <https://mochajs.org/>`__. A Mocha tesztek sorosan hajtódnak végre
lehetősget nyujtva a teszt esetek hibáinak kimutatására. 


Chai
----

A Chai könyvtár használatával könnyedén megvalósitható a tesztvezérelt fejlesztés,
szoftvertervezési folyamat, amely során teszteseteket irunk és ezeknek
megfelelően irjuk meg a kódot.
