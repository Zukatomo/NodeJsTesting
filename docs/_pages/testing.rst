Tesztelésben használt keretrendszerek
=======================================

A tesztelés során két keretrendszert használtam,
a `Mocha <https://mochajs.org/>`__ és `Chai <https://www.chaijs.com/>`__-t.

A tesztelés során a REST API néhány tulajdonságát fogom csupán tesztelni.
Ezek a tulajdonságok, az események lekérdezése különböző paraméterek szerint,
illetve új esemény hozzáadása és autentikáció.


Mocha
-----

A Mocha egy funkciókban gazdag, tesztelést segítő
keretrendszer, amely NodeJS futtatókörnyezetben működik, vagy
akár a böngészőben is `[1] <https://mochajs.org/>`__. A Mocha tesztek sorosan hajtódnak végre
lehetőséget nyújtva a tesztesetek hibáinak kimutatására. 



Chai
----

A Chai könyvtár használatával könnyen megvalósítható a tesztvezérelt fejlesztés,
szoftver tervezési folyamat, amely során teszteseteket írunk és ezeknek
megfelelően írjuk meg a kódot.
