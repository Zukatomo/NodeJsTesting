A szerver futtatása
--------------------

Az API szerver futtatása rendkivűl egyszerű, futattahtó bármilyen rendszeren
amely képes a NodeJS futtatására.

A fajlok letőltése után, a főkönyvtába lépve telepiteni kell az NPM csomagokat,
terminálban futtatva a következő parancsot:

.. code-block:: cmd

    npm install

A telepités befejezte után a szerver már készen is áll a használatra,
amelyet a következő paranccsal indithatunk:

.. code-block:: cmd

    node index.js

Ezt követően az API máris elérhető a következő cimen:
:literal:`\ \ http://localhost:3000`
