A szerver futtatása
--------------------

Az API szerver futtatása rendkívül egyszerű, futtatható bármilyen rendszeren
amely képes a NodeJS futtatására.

A fájlok letöltése után, a főkönyvtába lépve telepíteni kell az NPM csomagokat,
terminálban futtatva a következő parancsot:


.. code-block:: cmd

    npm install

A telepítés befejezése után a szerver már készen is áll a használatra,
amelyet a következő paranccsal indíthatunk:
    


.. code-block:: cmd

    node index.js

Ezt követően az API máris elérhető a következő cimen:
:literal:`\ \ http://localhost:3000`
